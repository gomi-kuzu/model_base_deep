# 『モデルベース深層学習と深層展開』勉強用リポジトリ
## 環境セットアップ
- 検証済みのホストはUbuntuのみ
- CLI版dockerを公式の方法に沿って入れている
### 立ち上げ
- docker compose v1の場合
```
docker-compose up -d
```
- docker compose v2の場合
```
docker compose up -d
```
